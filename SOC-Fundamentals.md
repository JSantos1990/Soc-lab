# 🛡️ SOC Fundamentals Overview

This document introduces the core concepts of a Security Operations Center (SOC) and the responsibilities of its analysts.

---

## 🧭 1. What Is a SOC?

A Security Operations Center monitors, detects, and responds to security events in real time.  
Its main functions are:

- Log collection and analysis  
- Alert investigation  
- Threat detection  
- Incident response  
- Reporting and documentation

## 👤 2. SOC Analyst Roles

### Tier 1 – Alert Analyst
- Monitors alerts  
- Performs triage  
- Escalates suspicious cases  

### Tier 2 – Incident Responder
- Investigates escalated alerts  
- Analyzes logs and evidence  
- Contains threats  

### Tier 3 – Threat Hunter
- Proactively searches for threats  
- Identifies patterns and anomalies
