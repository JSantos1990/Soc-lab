# Wireshark Analysis

This folder contains hands-on packet capture investigations performed using Wireshark.  
Each analysis focuses on identifying suspicious network behavior, understanding protocols, and recreating attacker activity from captured traffic.

Typical topics include:
- HTTP, DNS, SSH, and TLS inspection
- Identifying scanning or probing activity
- Detecting malicious payloads
- Extracting IoCs from PCAP files
- Reconstructing sessions and flows
